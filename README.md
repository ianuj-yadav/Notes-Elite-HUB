# NotesHub Elite - Complete Website

## Overview
NotesHub Elite is a comprehensive platform for students to buy and sell study materials. This package contains a complete multi page website with modern design and full functionality.

## Files Included
- index.html - Main landing page
- login.html - User login page
- signup.html - User registration page
- dashboard.html - User dashboard
- marketplace.html - Browse and buy notes
- publish.html - Upload and sell notes
- profile.html - User profile management
- style.css - Complete CSS styling
- script.js - JavaScript functionality

## Features
✅ Responsive design for all devices
✅ Complete authentication system
✅ User dashboard with analytics
✅ Advanced marketplace with filters
✅ File upload system for publishing notes
✅ User profile management
✅ Shopping cart functionality
✅ Search and filtering
✅ Modern UI/UX design
✅ Interactive animations
✅ Form validation
✅ Local storage for user sessions

## Setup Instructions
1. Extract all files to a folder
2. Open index.html in a web browser
3. Navigate through the website using the menu
4. All functionality is client-side and works offline

## Usage Notes
- This is a frontend demo with simulated backend functionality
- User sessions are stored in browser localStorage
- File uploads are simulated (files are not actually stored)
- All data is for demonstration purposes

## Customization
You can customize colors, fonts, and layouts by modifying the CSS variables in style.css:
- --primary-purple: Main brand color
- --primary-blue: Secondary brand color
- --font-family: Typography

For more advanced customization, modify the JavaScript in script.js to add real backend integration.
